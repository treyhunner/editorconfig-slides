EditorConfig

The Manifesto

  • You need a code style

    I don't care what style you use, but please pick a style and use it consistently.

  • Your code, your rules

    If I contribute code to your project I must obey the existing code style.

  • Let's not argue about tabs and spaces

    It's 2013. Your text editor should be handling this for you.

  • If you can automate it, you must automate it

    Tabs, spaces, whatever. Let the computer figure it out.

The Solution: EditorConfig

  1. Add an .editorconfig file

    Drop an .editorconfig file in your project. This specifies the code style for your files.

  2. Download an EditorConfig plugin

    Download an EditorConfig plugin for your text editor.

  3. Problem solved

    Really, that's it. You've just trained your editor to keep you in line.
